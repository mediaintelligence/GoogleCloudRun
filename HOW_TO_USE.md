# How to Use Claude-Gemini Assistant in VS Code

## ✅ Extension Successfully Installed!

The Claude-Gemini Assistant extension is now installed and ready to use in VS Code.

## 🚀 Quick Start

### Method 1: Command Palette
1. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
2. Type "Claude Assistant" to see all available commands
3. Select any command to run it

### Method 2: Keyboard Shortcuts
- **`Cmd+Shift+G`** (Mac) / **`Ctrl+Shift+G`** (Win/Linux): Start Gemini Workflow
- **`Cmd+Shift+C`** (Mac) / **`Ctrl+Shift+C`** (Win/Linux): Execute Claude Code with Context

### Method 3: Right-Click Context Menu
- Right-click in the editor to access Claude Assistant commands
- Right-click on folders in Explorer to analyze projects

### Method 4: Activity Bar
Look for the Gemini Assistant icon in the VS Code activity bar (left sidebar)

## 📋 Available Commands

### Core AI Features
- **Start Gemini Workflow**: Begin systematic AI-assisted development
- **Execute Claude Code with Context**: Run Claude with full project context
- **Analyze Project with Claude**: Deep analysis of your codebase

### Code Generation & Refactoring
- **Generate Code with AI**: Create new code based on requirements
- **Refactor Code with AI**: Improve existing code structure
- **Optimize Performance with AI**: Enhance code performance

### Debugging & Testing
- **Debug with AI**: Get AI assistance for debugging
- **Generate Tests with AI**: Automatically create test cases
- **Explain Code with AI**: Get detailed code explanations

### Session Management
- **Create New Work Session**: Start a new development session
- **Resume Previous Session**: Continue where you left off
- **Save Current Session**: Save your current work state
- **View Session History**: Browse past sessions

### Collaboration Features
- **Start Claude-Gemini Collaboration**: Use both AIs together
- **Collaborative Debug**: Debug with both AI models
- **Collaborative Refactor**: Refactor using combined AI insights
- **Compare AI Approaches**: See different AI solutions

### Recovery & History
- **Create Recovery Point**: Save current state for recovery
- **Restore from Recovery Point**: Restore previous state
- **Show Project Learning History**: View AI learning patterns
- **Review Last Claude Code Execution**: Check previous runs

## 🔧 First Time Setup

1. **Set API Key** (if using Gemini features):
   - Open VS Code Settings (`Cmd+,` or `Ctrl+,`)
   - Search for "Gemini Assistant"
   - Enter your Google Gemini API key
   - Get API key from: https://makersuite.google.com/app/apikey

2. **Configure Claude Code Path** (if needed):
   - Settings > Search "Claude Assistant"
   - Set "Claude Code Path" (default: "claude-code")

## 📁 Data Storage

Your work is automatically saved in:
`~/Library/Application Support/Code/User/globalStorage/mediaintelligence.claude-gemini-assistant/`

This includes:
- Sessions and recovery points
- Project analysis data
- AI interaction history
- Learned patterns
